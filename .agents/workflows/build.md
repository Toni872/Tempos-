# Build de producción

1. Generar build optimizado:
```bash
cd c:\Users\Green Republic\Desktop\Tempos\Frontend && npm run build
```

2. Previsualizar build localmente:
```bash
cd c:\Users\Green Republic\Desktop\Tempos\Frontend && npm run preview
```

El preview estará disponible en `http://localhost:4173`
