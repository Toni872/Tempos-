# Iniciar servidor de desarrollo

// turbo-all

1. Instalar dependencias si no existen:
```bash
cd c:\Users\Green Republic\Desktop\Tempos\Frontend && npm install
```

2. Iniciar servidor Vite:
```bash
cd c:\Users\Green Republic\Desktop\Tempos\Frontend && npm run dev
```

El servidor estará disponible en `http://localhost:5173`
